When using the shim annotation, be sure to do the right things as the list provided below.

1. Be sure to have the exact same argument type and arguments as the original C++ or C function.
2. You can manage how many arguments the function can have, but if an argument is not exist in the function,
it will not be a parameter that could be used for the function.
3. You can also change the function's name since both (Typescript, C, and C++) are different programming languages.
